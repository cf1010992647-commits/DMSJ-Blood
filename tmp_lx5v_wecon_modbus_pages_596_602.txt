10.9 Wecon Modbus protocol description
The currentWecon Modbus protocol description (special protocol) is modified based on the ModbusRTU protocol.
Therefore, 7-bitdata bits cannotbe used in the serial portparameter part.
The protocol of COM0 and COM1 can only use WECON Modbus protocol atpresentand cannotbe changed.
This protocol is fully compatible with Modbus RTU protocol, and the address is also the same as the defaultaddress of LX5V PLC's
Modbus RTU protocol.
The extended function is mainly used to communicate with PLCEDITOR.
